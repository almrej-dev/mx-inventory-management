  <name>Task 1: Create recipe server actions for CRUD and BOM retrieval</name>
  <files>src/actions/recipes.ts</files>
  <action>
Create `src/actions/recipes.ts` with "use server" directive, following the exact pattern from `src/actions/items.ts`:

**`createRecipe(rawData: unknown)`**
- `requireRole("staff")` guard (try-catch returning `{ error: { _form: ["Unauthorized"] } }`)
- Parse with `recipeSchema.safeParse(rawData)`, extract field errors on failure using the same reduce pattern as items.ts
- Call `checkCircularReference(parentItemId, childItemId)` for each ingredient. If any returns true, return `{ error: { _form: ["Circular recipe detected: this ingredient would create a cycle"] } }`
- Validate parent item exists, is FINISHED or SEMI_FINISHED, and has `deletedAt: null`
- Validate each child item exists, is NOT FINISHED type (only RAW_MATERIAL, SEMI_FINISHED, PACKAGING allowed as ingredients), and has `deletedAt: null`
- Use `prisma.$transaction` to:
  1. Delete any existing RecipeIngredient rows for this parentItemId (upsert pattern)
  2. `recipeIngredient.createMany` with ingredient data, converting `quantityGrams` to `quantityMg` using `gramsToMg()` from `@/lib/utils`
- `revalidatePath("/recipes")`
- Return `{ success: true }`

**`updateRecipe(parentItemId: number, rawData: unknown)`**
- Same as createRecipe but receives parentItemId explicitly. Same transaction pattern (delete old + create new).
- `revalidatePath("/recipes")` and `revalidatePath(\`/recipes/${parentItemId}\`)`

**`deleteRecipe(parentItemId: number)`**
- `requireRole("staff")` guard
- `prisma.recipeIngredient.deleteMany({ where: { parentItemId } })`
- `revalidatePath("/recipes")`
- Return `{ success: true }`

**`getRecipes()`**
- `requireRole("viewer")`
- Fetch all items that have recipe ingredients: `prisma.item.findMany({ where: { deletedAt: null, recipeIngredients: { some: {} } }, include: { recipeIngredients: { include: { childItem: { select: { id, name, sku, type, costCentavos, unitWeightMg } } } } }, orderBy: { name: 'asc' } })`
- For each recipe item, compute total cost using the included ingredient data (in-memory calculation, not calling calculateRecipeCost per item to avoid N+1):
  - For each ingredient: if PACKAGING, cost = `quantityPieces * childItem.costCentavos`; else cost = `Math.round(quantityMg * childItem.costCentavos / childItem.unitWeightMg)`
  - Sum all ingredient costs for the recipe total
  - Note: This handles only direct (single-level) cost. For recipes with SEMI_FINISHED ingredients that have their own recipes, show cost with a note. Full multi-level cost is shown on the detail page via explodeBom.
- Return `{ recipes: RecipeListItem[] }` with shape: `{ id, sku, name, type, ingredientCount, totalCostCentavos, costPesos }`

**`getRecipe(parentItemId: number)`**
- `requireRole("viewer")`
- Fetch the parent item + its recipe ingredients with child items included
- Return the recipe data or null if not found
- Convert quantities to display units for the edit form: `quantityMg` -> grams using `mgToGrams()`

**`getRecipeBom(parentItemId: number)`**
- `requireRole("viewer")`
- Call `explodeBom(parentItemId)` from `@/lib/bom`
- Call `calculateRecipeCost(parentItemId)` from `@/lib/bom`
- Return `{ bom: BomLine[], totalCostCentavos: number, totalCostPesos: string }`
- Catch errors (e.g., circular reference) and return `{ error: string }`

**`getItemsForIngredientSelect()`**
- `requireRole("viewer")`
- Fetch all active items that can be ingredients: `prisma.item.findMany({ where: { deletedAt: null, type: { in: ['RAW_MATERIAL', 'SEMI_FINISHED', 'PACKAGING'] } }, select: { id, name, sku, type, unitWeightMg, costCentavos }, orderBy: { name: 'asc' } })`
- Return `{ items }` -- used by the recipe form ingredient dropdown

**`getItemsForParentSelect()`**
- `requireRole("viewer")`
- Fetch all active FINISHED and SEMI_FINISHED items: `prisma.item.findMany({ where: { deletedAt: null, type: { in: ['FINISHED', 'SEMI_FINISHED'] } }, select: { id, name, sku, type }, orderBy: { name: 'asc' } })`
- Return `{ items }` -- used by the recipe form parent product dropdown
